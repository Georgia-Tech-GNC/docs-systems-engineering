# Codex Instructions

When editing this repository, follow `docs/writing-style.md`.

Keep `README.md` as a navigation page. Put process definitions in the matching file under `docs/`, and put reusable artifact skeletons under `templates/`.

Do not duplicate definitions across files. If a concept belongs in more than one place, define it once and link to it.

Before adding a new top-level document, confirm that it has a distinct purpose that is not already covered by an existing document.
